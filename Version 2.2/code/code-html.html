📖 Introduction :

Le HTML (HyperText Markup Language) est le langage de base des pages web.
Il sert à structurer les contenus : titres, paragraphes, images, liens...

Exemple de structure HTML de base

🔍 Survole moi 👇

🧠 Explication rapide :

  • <!DOCTYPE html>: indique que c’est une page HTML5
  • <html>: tout le contenu HTML est à l’intérieur
  • <head>: contient des infos invisibles (titre, encodage...)
  • <title>: le nom affiché dans l’onglet du navigateur
  • <body>: le contenu visible de la page
  • <h1>: un titre principal
  • <p>: un paragraphe
